## Knowledge Evidence 机制（关键细节）

### 公式化

$$Y = f(Q, D, K | \theta)$$

其中 K 是 **external knowledge evidence**（外部知识证据）

### 四类 Knowledge Evidence

| 类别 | 说明 | 示例 |
|------|------|------|
| **Numeric Reasoning** | 数学计算知识 | MINUS, ADDITION, DIVISION, MULTIPLY, 百分比、公式 |
| **Domain Knowledge** | 领域专业知识 | 金融指标（ROI, net income）、业务规则 |
| **Synonym Knowledge** | 同义词映射 | 不同表达指同一概念 |
| **Value Illustration** | 值描述 | "center" → "pos = C" in professional_basketball |

### Database Description File

BIRD 提供 **database description file** 作为：
- Schema DDL 描述
- 表/列含义解释
- 外键关系说明
- 值域约束

## 与本研究 Ontology 的对比

| 维度 | BIRD Knowledge Evidence | 本研究 Ontology |
|------|------------------------|----------------|
| **形式** | 非结构化文本片段 | 结构化图谱 |
| **粒度** | 问题级（每个问题可能有 evidence）| 数据库级（覆盖整个 DB）|
| **推理** | 无推理机制 | 元路径约束推理 |
| **召回** | 数据集提供，无筛选 | 联合召回 + 最小完备集 |
| **构建** | 人工标注 | LLM 半自动 |
| **类型** | 4 类固定 | 概念+属性+关系+术语（灵活）|

### 关键区别

BIRD 的 knowledge evidence 是 **非结构化文本**，LLM 只能阅读理解，不能做结构化推理。本研究 Ontology 是 **结构化图谱**，支持元路径约束和多跳推理。
